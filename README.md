##NOMBRE
NOMBRE es una aplicación de citas por la cual los usuarios pueden entablar conversaciones con otros usuarios con el fin de conocerse y poder entablar relaciones amorosos. Difiere de las aplicaciones de citas convenciales en cuanto a que la aplicación tiene juegos cooperativos en los cuales los usuarios interactúan entre ellos, para comprobar su compatibilidad, etc. Con estos juegos se pueden ganar puntos con otros usuarios, que servirán para desbloquear la funcionalidad del chat, de esta manera, evitamos las interacciones efímeras y pasajeras, para fomentar las duraderas y reales.
