Que queremos que tenga nuestro proyecto en el formato

Como usuario, quiero ____ para _____



Como usuario, quiero tener una interfaz gráfica llamativa para que sea agradable usar la aplicación.

Como usuario, quiero tener una interfaz gráfica intuitiva, para poder moverme con facilidad sin tener que recordar dónde están las cosas.

Como usuario, quiero crear un perfil para controlar todos mis datos tales como foto, nombre, hábitos o intereses.

Como usuario, quiero que mi perfil tenga una contraseña para poder poner la cuenta en distintos aparatos.

Como usuario, quiero salir de mi perfil sin necesidad de desinstalar la aplicación para más seguridad.

Como usuario, quiero tener mis chats y la opción de jugar en la pantalla de inicio para que sea lo más fácil posible acceder a ellos.

Como usuario, quiero ver perfiles de otros usuarios.

Como usuario, quiero buscar perfiles según distintos filtros.

Como usuario, quiero reportar a otros usuarios para evitar casos de acoso o violencia verbal.

Como usuario, quiero eliminar chats pasados para que dejen de estar ahí.

Como usuario, quiero buscar entre mis chats para encontrar a la persona que busco.


Como usuario 1, quiero jugar en línea con el usuario 2 para ver lo compatibles que somos.

Como usuario 1, quiero ganar puntos jugando con el usuario 2, para avanzar.

Como usuario 1, quiero jugar a juegos cooperativos para conocer más al usuario 2.

Como usuario 1, quiero avanzar lo suficiente en el juego con el usuario 2 para poder hablar y quedar con él. 

Como usuario 1, quiero chatear con otros usuarios una vez hayamos avanzado en el juego para poder conocernos mejor.

Como usuario, quiero variedad de juegos y de niveles para que no sea demasiado repetitivo al jugar con distintas personas.

Como usuario, quiero salir de los juegos para dejar de jugar.

Como usuario, quiero invitar a la gente para jugar juntos.

Como usuario 1, quiero emitir una solicitud al usuario 2 para cambiar de juego.

Como usuario 1, quiero aceptar o rechazar la solicitud del usuario 2 de cambiar de juego para seguir jugando.



Como moderador, quiero ver los chats de los usuarios reportados para aplicar medidas de ser necesario.


Como desarrollador, quiero añadir nuevos juegos para crear una experiencia diferente.

Como desarrollador, quiero que la aplicación necesite el menor mantenimiento posible, para facilitar el desarrollo en el futuro.
